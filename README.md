# Active Directory IT Support Lab – Windows Server 2022

This repository documents my hands-on journey learning and practicing Active Directory administration using a Windows Server 2022 lab environment.

Instead of only studying theory, I decided to build my own domain from scratch and simulate real IT Support tasks that happen in everyday work environments.

---

## 💻 Lab Environment

- Windows Server 2022 (Desktop Experience)
- Active Directory Domain Services
- VMware Workstation
- Domain: mishel.local

This lab is built as a safe space to learn, make mistakes, troubleshoot, and understand how identity and access management works behind the scenes.

---

## 🧠 What I'm Practicing

Throughout this project I’m focusing on skills commonly used in IT Support and system administration roles:

- Setting up a Domain Controller
- Configuring static IP and DNS
- Designing Organizational Units (OU)
- Creating and managing users
- Working with security groups and permissions
- Exploring Group Policy basics
- Troubleshooting common configuration issues
